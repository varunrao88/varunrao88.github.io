Are Developing Countries really Developing ?

This project aims to look at the world and identify the progress being made by countries across the globe. The question that we try to answer is that are developing countries showing a trend towards becoming fully developed or if it's just a dream

About the Visualization

About the Data

The data set has been picked up from the World Bank indicators that are freely provided. The data provides a year by year breakdown of how countries across the world are faring on various parameters. The World Bank provides an exhaustive list of parameters, for the purpose of this project I have selected a subset of these.

The Visualization Structure

I chose the Interactive Slideshow approach. Following this page you have the option of viewing the data on a macroscopic scale to analyse the urbanization of countries from 1980 to 2014. You can then interact with the data and select countries which you want to take a deeper look into. You can also select any one the presets which is a group of countries that have already been shortlisted.

Scenes

The scenes have been kept uniform in order to add a level of coherence for the user. The graph is placed on the left side of the screen with information regarding the scenes, parameters, annotations and triggers on the right. Please look out for the navigation markers on the right to proceed.
